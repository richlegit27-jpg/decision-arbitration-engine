\# Nova Landing Page Layout



\## Page Order



1\. Top Navigation

2\. Hero Section

3\. Product Summary

4\. Feature Highlights

5\. How Nova Works

6\. Call To Action

7\. Footer



\## 1. Top Navigation

Contents:

\- Nova logo or name

\- Login button

\- Get Started button



Goal:

Give the page an immediate product feel and clear entry points.



\## 2. Hero Section

Contents:

\- Main headline

\- Short subheadline

\- Primary button

\- Secondary button



Goal:

Immediately explain what Nova is and what the user should do next.



\## 3. Product Summary

Contents:

\- Short paragraph explaining Nova

\- Focus on persistent AI workspace, memory, attachments, and local control



Goal:

Give the user a simple understanding of the product without overwhelming them.



\## 4. Feature Highlights

Contents:

\- Persistent Conversations

\- Built-in Memory

\- File Attachments

\- Workspace Awareness

\- Local Control



Goal:

Show the main reasons Nova is useful.



\## 5. How Nova Works

Contents:

\- Open Nova

\- Start a conversation

\- Attach files or add context

\- Continue later

\- Let Nova remember what matters



Goal:

Make the flow feel simple and understandable.



\## 6. Call To Action

Contents:

\- Strong closing line

\- Start Chatting button

\- Login button



Goal:

Push the user toward action.



\## 7. Footer

Contents:

\- Nova

\- Local AI Workspace

\- Version 0.1



Goal:

Close the page cleanly and make the product feel complete.



\## Visual Rules

\- Clean layout

\- Strong spacing

\- Clear sections

\- Dark polished style

\- No clutter

\- No overloaded text



\## Build Rule

Do not overcomplicate the first landing page.

Make it clean, clear, and real first.
