\# Nova Product Scope



\## What Nova Is

Nova is a local AI chat application with memory, attachments, and workspace awareness.



\## Launch Goal

Ship a stable local version that feels clean, usable, and reliable.



\## Core Features Required

\- Chat

\- Streaming responses

\- Memory panel

\- Chat sidebar

\- File attachments

\- Local runtime storage



\## Features Not Required For Launch

\- Mobile app

\- Desktop packaging

\- Advanced deployment

\- Extra integrations

\- Big redesigns



\## Launch Standard

Nova should:

\- open

\- send messages

\- show responses

\- keep chats

\- use memory

\- handle attachments

\- avoid obvious UI breakage



\## Rule

No major new features until the core launch version feels stable.
