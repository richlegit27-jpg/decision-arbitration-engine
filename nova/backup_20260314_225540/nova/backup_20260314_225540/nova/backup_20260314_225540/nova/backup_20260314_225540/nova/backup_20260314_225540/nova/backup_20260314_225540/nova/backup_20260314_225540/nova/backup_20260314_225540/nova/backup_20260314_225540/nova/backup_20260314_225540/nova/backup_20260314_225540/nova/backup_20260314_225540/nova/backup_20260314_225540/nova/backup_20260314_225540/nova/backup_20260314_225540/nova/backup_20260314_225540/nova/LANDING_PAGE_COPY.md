\## Product Summary



Nova is a persistent AI workspace that remembers context, understands your files, and keeps conversations organized across sessions.



Instead of starting over every time you open a chat, Nova builds continuity. Conversations, memory, and attachments work together so the AI becomes part of your workflow rather than a disposable tool.



Nova runs locally and focuses on clarity, control, and productivity.





\## Feature Highlights



\### Persistent Conversations

Your chats stay organized and accessible. Continue where you left off without losing context.



\### Built-in Memory

Nova can remember important information so conversations become smarter over time.



\### File Attachments

Attach documents, images, or files directly into conversations.



\### Workspace Awareness

Nova can reference project files and maintain workspace context while you work.



\### Local Control

Designed to run locally with clear data storage and predictable behavior.





\## How Nova Works



1\. Open Nova  

2\. Start a conversation  

3\. Attach files or add context  

4\. Continue the same conversation later  

5\. Let Nova remember what matters
