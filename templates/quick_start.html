Nova Ultimate 2026 Quick Start

1. Launch

Open PowerShell in the Nova folder and run:

.\launch_nova_portable.ps1

Backend and frontend start automatically, browser opens to the chat interface.

2. Sidebar & Panels

Use the sidebar toggle ☰ to collapse or expand. Each panel can be opened/closed:

  • Memory: Shows chat history
  • Attachments: Upload / download / delete files
  • Exports: Download memory as JSON
  • Voice: Record audio / play TTS AI
  • Images: Preview uploaded images
  • New Chat: Create or switch sessions

3. Chat Composer

Type a message and click Send. AI replies stream with typing indicator.

  • Stop: Cancel streaming
  • Theme: Toggle dark/light
  • Clear Memory: Clears current session memory
  • File input: Upload files or images

4. Voice & TTS

Voice panel allows:

  • Start Recording / Stop Recording → auto-added to attachments
  • Play AI TTS → plays latest AI message

5. Multi-Session Chat

Create or switch between sessions in the New Chat panel. Each session keeps its own memory, attachments, and images.

6. Demo Mode

Demo mode automatically opens all panels and populates sample messages and files.

7. Stopping Nova

Use the printed backend PID to stop the server:

Stop-Process -Id <PID> -Force
