What is Nova?

Nova is an AI workspace being built to combine chat, project memory, file handling, planning, voice, public pages, and account/billing systems into one product.

Is Nova fully launched?

Not yet. Nova is moving toward public launch. The public website, contact capture, CRM-lite lead flow, billing explanation pages, and release checks are stable. Some payment and advanced account features are still planned.

What is live right now?

The current stable public stack includes the landing page, public site pages, FAQ, contact and Early Access lead capture, CRM-lite admin leads, billing clarity copy, Railway release checks, and the core Nova app foundation.

What is still planned?

Planned items include Stripe checkout, production payment collection, selectable model tiers, model-based pricing, more detailed account controls, and deeper usage history for users.

Can I pay for Nova today?

Nova currently explains billing, credits, usage, and planned payments clearly, but live Stripe payment collection is still planned. Early Access users can register interest before full checkout is enabled.

How do credits work?

Credits are Nova’s way of explaining usage. Different AI features and future model tiers may use credits differently. The goal is to make usage easier to understand before users spend money.

Will Nova support different AI models?

Yes, model-tier support is planned. The current public billing page explains that different models may cost different amounts in the future, so users are not surprised when model selection becomes available.

Is Nova only a chatbot?

No. Chat is the core interface, but Nova is being built as a broader workspace with sessions, planning, file support, memory, voice features, public pages, lead capture, and account systems.

Is my information sold to advertisers?

Nova’s public product direction is based on user trust, clear billing, and direct product value. The public site should avoid vague promises and explain privacy and billing plainly.

How do I get Early Access?

Use the Early Access form to send your name, email, and interest. Nova stores the lead so it can be reviewed from the admin lead flow.
