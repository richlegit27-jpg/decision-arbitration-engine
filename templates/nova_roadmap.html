Nova Roadmap

Nova is moving toward a more continuous AI workspace.

Nova is moving toward a workspace that carries the thread of a project forward, understands what matters now, and helps turn long conversations into useful next actions.
Now

Hold the working thread

Strengthen the connection between conversation, project state, memory, files, and the work a user is actively trying to move forward.

  • Sharper project-state recall
  • Cleaner continuity across sessions
  • Better file and attachment understanding
  • More useful next-action judgment
Next

Turn context into momentum

Make Nova better at recognizing the current objective, separating signal from noise, and helping a user move from discussion into organized execution.

  • Clearer project priorities
  • Stronger planning and execution flow
  • Better progress and decision visibility
  • Safer model and usage controls
Later

Deepen project control

Expand the workspace around long-running work so important decisions, artifacts, actions, and project history stay easier to find and use.

  • Richer project command views
  • Deeper artifact organization
  • More capable tool-driven actions
  • Stronger continuity across devices

Product direction

The goal is not more noise. It is more continuity.

Nova is being shaped around the parts of serious work that usually get scattered: the goal, the current state, the decisions already made, the files involved, and the next useful move.
Conversation Keep the current objective and important context in view.
Project state Understand what changed, what is protected, and what still needs work.
Execution Move from goals to plans, actions, results, and clearer follow-through.
Workspace Bring sessions, files, memory, and project history into one useful thread.
