Nova FAQ

The important questions. Straight answers.

Nova is built for projects that outlive a single chat. Here is how project context, memory, files, access, and billing fit into the workspace.
01

What is Nova?

Nova is an AI workspace for projects that take more than one conversation. It is designed to keep project context, files, sessions, and important decisions connected as the work moves forward.

02

How is Nova different from a normal chatbot?

Nova is organized around the project instead of only the current prompt. Sessions, files, project context, and follow-up work are meant to stay connected so you can continue the work instead of rebuilding it.

03

Is Nova desktop-first?

Yes. Nova is designed around a full project workspace where sessions, files, memory, and project context can stay visible and organized. The focus is continuity across real work, not squeezing a large project into a one-prompt interface.

04

Does Nova remember projects?

Nova uses project memory and session continuity to carry useful context and previous decisions forward. The goal is to reduce repeated explanations while keeping important project facts available for the work that follows.

05

Can Nova handle uploads?

Yes. Nova can work with supported file and image attachments so project material can become part of the conversation instead of forcing everything through a typed prompt.

06

How does billing work?

Nova is currently preparing its access and billing experience. The billing page explains the current account and credit path. Sending an Early Access request does not start a paid subscription or charge a card.

07

Will Nova support different AI models?

Nova is being designed to support controlled model choices with different usage costs. Model availability, credit cost, and paid access should be shown clearly before a user chooses a paid option.

08

How do I request access?

Use the Request Access form and tell us what you are building, what kind of work needs continuity, and where your current AI workflow keeps losing the thread.
