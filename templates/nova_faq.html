FAQ

Questions about Nova, answered clearly.

Nova is an AI project command center built around long-running work: memory, sessions, uploads, voice, billing credits, and a stable desktop-first workspace.
01

What is Nova?

Nova is an AI workspace for projects that take more than one chat. It is designed to help keep context, files, sessions, and project decisions together.

02

How is Nova different from a normal chatbot?

Normal chatbots usually focus on one prompt at a time. Nova is being built around the full project: memory, sessions, uploads, billing, and practical execution.

03

Is Nova desktop-first?

Yes. The desktop workspace is the stable product lane right now. Mobile can be rebuilt cleanly later after the public site and desktop experience are stronger.

04

Does Nova remember projects?

Nova is built with project memory and session continuity in mind, so useful facts and previous decisions can carry forward instead of disappearing.

05

Can Nova handle uploads?

Yes. Uploads are part of the workspace direction, so Nova can work from files and images instead of relying only on typed prompts.

06

What is the billing system?

Nova has a local credit foundation for tracking usage. The dashboard can grow into real model tiers, plan limits, and account controls later.

07

What model tiers are planned?

The future plan is to support different model levels with different credit costs, controlled by the backend rather than only the user interface.

08

What should be built next?

The safest path is public product polish, billing clarity, desktop stability, and then a clean mobile rebuild on a separate branch.

Still building

Nova is moving from prototype to product.

The public pages explain what Nova is, the billing dashboard shows account status, and the desktop workspace remains the strongest place to keep building.
