What Nova is

Nova is your local AI workspace. It can chat, remember project context, manage sessions, analyze uploads, help with coding, and run simple execution missions through auto-plan.

Chat

  • Type a message and press Enter or tap Send.
  • Use k, ok, next, or continue when Nova is waiting for the next step.
  • Use summarize this to get a short summary of the current context.

Uploads / Images

  • Tap Attach to upload an image or file.
  • Ask what is this image after attaching an image.
  • Ask summarize this file after attaching a document.
  • If the answer says only “Attachment received,” the attachment reached the chat but analysis routing may need checking.

Sessions

  • New starts a fresh chat.
  • Sessions opens saved chats.
  • Sessions are scoped to the logged-in local user.
  • Duplicate empty sessions are pruned automatically.

Memory

  • Memory shows saved project/context items.
  • Memory is used to remember project focus, blockers, active files, and recent state.
  • Direct endpoint: /api/memory

Tools / Execution

  • Start a mission with auto-plan <goal>.
  • Example: auto-plan build a simple password generator
  • Advance one step with next or run step.
  • Run the whole mission with run all.
  • Cancel with cancel or stop.

Quick buttons

  • Go means continue from where we left off.
  • Sum means summarize the current situation.
  • Fix means improve/clean up the current thing.
  • Next means move to the next step.

Known note

If run all returns web/news results, execution command routing is being stolen by the web route. That is a routing priority bug, not a user error.
